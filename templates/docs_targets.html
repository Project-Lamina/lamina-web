Targets and backends

Target names use architecture_os. The driver checks the name first; the selected output mode then reports whether the backend supports it.

Published rustdoc platforms

The docs.rs build for Lamina 0.0.8 includes Linux, Windows, and Apple platforms. Those rustdoc platforms are separate from the target names accepted by the compiler driver.
