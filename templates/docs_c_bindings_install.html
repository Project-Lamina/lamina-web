C bindings source preview

Build the C bindings

lamina-c is implemented in the development checkout, but public archives and the generated lamina.h header are not published yet. Use this page to inspect or test the source preview.

Not a public install yet

Do not depend on the current ABI from released software. The first public install will ship versioned archives, a reviewed header, checksums, and packaged smoke tests.

Current progress

Stable cratelamina-c exists and builds as a static library and shared library.

AOT APIOwned builder handles, module serialization, assembly compilation, buffers, and status codes are implemented.

Nightly JITThe optional JIT module builds behind --features nightly.

Public headerlamina.h generation and review are still pending.

Release bundlesGitHub Release archives and checksums are planned but not published.

Build from a source checkout

The stable source-preview build exposes AOT assembly generation. These commands build the library for contributors; they do not install a finished public SDK.

$ git clone https://github.com/SkuldNorniern/lamina.git
$ cd lamina
$ cargo build -p lamina-c --release

Apple Silicon macOS

The aarch64_macos compiler path has received partial testing. Treat macOS C binding builds as preview work until release coverage is broader.

Build the nightly JIT extension

JIT is not part of the stable C surface. Build with the nightly feature only when testing runtime compilation.

$ cargo build -p lamina-c --release --features nightly

The published SDK will keep JIT declarations in an additive lamina_nightly.h header. Stable programs should include only lamina.h.
