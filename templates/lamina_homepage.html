Compiler pipeline

From SSA IR to a linked binary.

Each stage is explicit, inspectable, and small enough to reason about.

01

program.lamina

Parse Lamina IR

Read a typed, block-based SSA representation with functions, globals, tuples, pointers, and explicit memory operations.

02

module.lumir

Lower to MIR

Move architecture-neutral intent into a machine-oriented representation ready for transformation and register work.

03

-O0 ... -O3

Run transforms

Apply CFG simplification, constant folding, dead-code removal, copy propagation, inlining, and strength reduction.

04

target.s / target.wat

Generate target code

Select instructions, allocate registers, build frames, and honor the selected target ABI without an external backend.

05

native executable

Assemble and link

Use Lamina's RAS and Weld toolchain path or choose familiar platform tools when your build needs them.

A frontend-friendly IR

Small surface. Serious backend.
